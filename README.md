#  Smart Word Counter Extension

A professional Chrome/Edge browser extension that shows word, letter, and character count of selected text near the cursor in real time.

---

##  Features

- Shows word count
-  Shows total letters
-  Characters (with spaces)
-  Characters (without spaces)
-  Modern popup UI
-  Works on most websites
-  Cursor-based positioning

---

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome/Edge Extension API (Manifest v3)

---

## Installation Guide

1. Download or clone this repository.
2. Open Chrome/Edge and go to:
   ```
   chrome://extensions/
   ```
3. Enable **Developer Mode**.
4. Click **Load Unpacked**.
5. Select the project folder.

---

##  How It Works

1. Select any text on a webpage.
2. A stylish popup appears near the cursor.
3. Displays text statistics instantly.
